Lets Play GAME

Follow these steps:

  1. Open Google Chrome and navigate to the page where the game is hosted (e.g., http://localhost:4001).
  2. Open the Developer Tools by right-clicking anywhere on the page and selecting "Inspect" or by pressing Ctrl + Shift + I (or Cmd + Option + I on macOS).
  3. In the Developer Tools window, navigate to the "Network" tab.
  4. With the Network tab open, make sure the "Preserve log" option is checked. This ensures that network requests are not cleared when you navigate to a new page.
  5. Now, you're ready to send a POST request to the server. Follow these steps:
    1. Click on the "Console" tab within the Developer Tools.
    2. Type the following JavaScript code in the console to send a POST request:

fetch('http://localhost:4001', {

method: 'POST',

body: 'rock'

})

.then(response => response.text())

.then(console.log)

.catch(console.error);
